# Advent of Code 2025

My solutions for the [Advent of Code 2025](https://adventofcode.com/2025) challenges.

## Structure

- `day1/`: Solution for Day 1
- `day2/`: Solution for Day 2
- `day3/`: Solution for Day 3

## Running the code

Ensure you have Python installed.

1.  Activate the virtual environment:
    ```bash
    source .venv/bin/activate
    ```
2.  Run a specific day's solution (example):
    ```bash
    python day1/solution.py
    ```
